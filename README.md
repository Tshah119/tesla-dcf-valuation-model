Tesla DCF Valuation Model
Overview

This project builds a Discounted Cash Flow (DCF) model to estimate the intrinsic value of Tesla, Inc. using projected free cash flows and the weighted average cost of capital (WACC).

Objective

The goal of this model is to estimate Tesla’s intrinsic value by forecasting future cash flows and discounting them back to present value using standard financial modeling techniques.

Model Components

• Revenue forecasting
• Operating margin projections
• Free cash flow calculations
• Weighted Average Cost of Capital (WACC) estimation
• Terminal value calculation using the Gordon Growth Method
• Sensitivity analysis

Key Assumptions

• Forecast Period: 5 Years
• Risk-Free Rate: 4.95%
• Beta: 1.71
• After-Tax Cost of Debt: ~4.5%

Tools Used

Microsoft Excel

Purpose

This model was created as an academic project to demonstrate financial modeling and valuation skills commonly used in corporate finance, equity research, and investment analysis.
